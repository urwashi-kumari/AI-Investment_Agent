# AI-Assisted Development Log

## Project
InvestIQ AI – AI-Powered Investment Research Platform



## Objective

The objective of this project was to build a full-stack web application that helps users analyze publicly listed companies using Artificial Intelligence and live financial data. The application provides financial health analysis, SWOT analysis, risk assessment, confidence score, and investment recommendations through a simple and interactive dashboard.

During the development of this project, I used ChatGPT as a learning and development assistant to understand new concepts, debug errors, improve the project structure, and prepare documentation. Google Gemini 2.5 Flash was integrated into the application using LangChain to generate AI-powered investment analysis.



## AI Tools Used

### ChatGPT (OpenAI)

I used ChatGPT throughout the development process as a technical learning assistant. It helped me understand concepts, discuss implementation approaches, debug issues, improve code quality, and solve deployment problems.

### Google Gemini 2.5 Flash

Google Gemini was integrated into the application through LangChain to generate investment reports based on live financial data and recent web search results.



## Development Process

### Project Planning

Before starting development, I explored different project ideas with ChatGPT. After discussing several possibilities, I decided to build an AI-powered investment research platform.

During this stage, ChatGPT helped me:

- Finalize the project idea.
- Select React for the frontend.
- Choose Node.js and Express.js for the backend.
- Understand how LangChain works with Google Gemini.
- Decide to use Finnhub for live stock market data.
- Use Tavily Search API for recent company information.
- Plan the overall folder structure of the project.

This planning helped me understand how different technologies could work together in a full-stack application.



### Backend Development

While building the backend, I used ChatGPT to understand how to organize an Express application using controllers, services, routes, and configuration files.

It also helped me understand:

- REST API development
- Error handling
- API validation
- Environment variable management
- Modular project structure

I implemented the backend based on these discussions and tested every feature during development.



### AI Integration

Integrating Google Gemini was one of the most interesting parts of the project.

I used ChatGPT to understand:

- LangChain configuration
- Prompt design
- Calling Gemini from the backend
- Parsing structured JSON responses
- Handling invalid AI responses

This helped me successfully integrate AI into the application.



### External API Integration

The project uses multiple external APIs.

During development, I integrated:

Whenever I encountered API-related errors or response issues, I used ChatGPT to understand the problem and identify possible solutions before implementing and testing them.

One of the biggest learning experiences during this project was debugging.
